# Ejercicios de POO en Kotlin

Este proyecto contiene cinco ejercicios prácticos de Programación Orientada a Objetos en Kotlin.

## Estructura

- `src/main/kotlin/Main.kt`: punto de entrada y demostración general.
- `src/main/kotlin/ejercicio1/Libro.kt`: clase `Libro`.
- `src/main/kotlin/ejercicio2/ProductoInventario.kt`: encapsulamiento con stock y precio.
- `src/main/kotlin/ejercicio3/Vehiculo.kt`: herencia con `Vehiculo`, `Auto` y `Motocicleta`.
- `src/main/kotlin/ejercicio4/MetodoPago.kt`: polimorfismo con métodos de pago.
- `src/main/kotlin/ejercicio5/Biblioteca.kt`: mini sistema de biblioteca.
- `evidencias/`: salidas de ejemplo para cada ejercicio.
